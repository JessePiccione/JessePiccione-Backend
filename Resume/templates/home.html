      Hi, my name is Jesse Piccione. I am an aspiring software developer, and the purpose of the application you are currently viewing is to showcase my skills in maintaining and executing the best software development practices. Shortly after high school, my life took a different turn when I found out that I was going to be a father. Nevertheless, I was still able to take on the role of father, provider, and student. In college, I participated in many project groups involving development in Java, C, C++, Python, Node.js, and Dart, just to name a few. My senior engineering project was an application called Chirp, a cross-platform mobile application designed using Google's Flutter Framework to power a machine learning model that makes predictions of birds in a present audio clip.

      During my last year of college, I was able to participate in the Spencer's E-commerce Java Developer Internship program, where I gained valuable insight into the software development industry. It is also where I had the privilege of designing and implementing an e-commerce application known as ItemApp. ItemApp is a Maven-managed Java Spring Boot application that performs statistical analysis of e-commerce site data and serves as a site service management tool.

      Post-college, I decided to participate in the Meta Professional Back-end Developer Program, which uses the very same technologies that are in play in the Single Page Web Application before you. Make no mistake, the application before you is not in any way, shape, or form a traditional website. A traditional website navigates from page to page and has a back-trackable user experience. Prior to developing this application, I had an idea to make a project that not only shows my skills and know-how but also displays them. That is what led to the app that I am calling JessePiccione.com otherwise known as my personal eResume.
