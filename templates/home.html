Why Software Engineering?

      My passion for software engineering began when I was in High School. I was always under the assumption that I was going to do something involving math. When I walked into my first computer programming class, I was not thinking that 'software engineer' would be the career path that I would be pursuing. I was told by a friend that I would love the class because I was into computers and had a math mindset for that kind of stuff. I take the class and within two weeks the teacher puts me into the College Level AP CS class where I went on and was one of two kids in my class that got a perfect score on the AP CS A exam. After that, I had some free time and was able to join a group in the next level class called Advanced Software Engineering and participate in the design and development of an application called shopping simulator. Shopping Simulator is a computer simulation of a grocery store built using the Unity Game Engine. My team was able to bring this to the 2018 Steam tank regionals and take home the Win. Although, what truly made me fall in love with software was this feeling of building an application. the feeling of being able to solve problems with software was my dream.
  • C/C++ - the main programming language behind the aplication and gaming engine.

Yes Software Engineering!

      In college, the school I obtained my ascociates in was based primarily in Java. This out me at an advantage given my prior experience in Highschool. When I finished my Ascociates, I then went to Oregon State University. In hindsight, this was my biggest challenge. Oregon State University, does not use Java for the majority of their computer science classes, I was now presented a challenge of working with 3 different langauges, NodeJs, Python, and Dart. At Oregon State I developed many applcations. I was one of the creators of a cross platform mobile application called Chirp. Chirp is a mobile application developed using dart and Google's Flutter framework. It is a powerful tool that can easily develop a single application that can be ran on several different platforms or operating systems. The purpose and main functional requirement of the Chirp application is to provide a connection to a bird sound prediction machine learning model. This project stands out to me the most because it invovled more than just coding problems. My team and I were able to go through every stage of the software development life cycle where we got to design, develop, and maintain a real world application that was sponsered by a client.
  • Dart - the programming language used to create an application of Flutter.
  • Flutter - the framework used to bring a standard to IOS and Android Devices.
  • AWS - the platform that hosts the backend of chirp as well as the ML Model.
  • NoSql - the backend database used to collect data for the machine learning model.
  • Python with Tensor Flow - the main language and library behind the machine learning model.

What is this? - JessePiccione.info

      Post College and internship, I have been looking for a way to combine my passion with my job search. That is where I came up with the idea for the application that is before you called JessePiccione.info. JessePiccione.info looks, feels and moves like a personal eResume. It is by no means a traditional website, but rather is fullstack web application that was designed and developed using Pythons Django Framework, a Bootstrap Frontend and a MySql database backend. All of the information stored on this application is controlled outside of the actual code. This simply means at anytime, with the right credentials the visible data can be changed and edited without restarting the application. Another key feature is the application also supports the ability to send a messsage to the admin using the twilio api. Please note the database also records the relavent message that was sent to serve as a fallback incase the twilio api fails. In the event a bug is discovered please report it by submitting a message, or contact me directly using the information below. I hope you enjoy this application it has been a journey to make it.
